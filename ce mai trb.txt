Crearea și ștergerea de elemente HTML:


Codul JavaScript verifică și implementează o serie de funcționalități conforme cu lista specificată:

    Modificarea stilului unui element: Este exemplificată prin schimbarea culorii fundalului pentru elementul cu id-ul "nav" și modificarea proprietății "borderColor" a unui buton la intervale regulate de timp.

    Manipularea DOM-ului: Selectarea elementelor se face folosind metoda document.getElementById, document.getElementsByClassName și document.querySelectorAll, după care se aplică diferite acțiuni asupra lor, cum ar fi adăugarea sau eliminarea unei clase CSS sau modificarea atributelor.

    Crearea și ștergerea de elemente HTML: Acest aspect nu este prezent în codul furnizat.

    Folosirea și modificarea evenimentelor generate de mouse și tastatură: Sunt folosite evenimente precum "click", "submit" și "keydown" pentru a gestiona interacțiunile utilizatorului.

    Folosirea setTimeout sau setInterval: Se utilizează funcția setTimeout pentru a amâna anumite acțiuni și pentru a opri anumite animații după un interval de timp dat.

    Folosirea localStorage: Nu este prezentă în codul furnizat.

    Folosirea a cel puțin unei metode din clasele: Math, Array, String, Date: Se folosește clasa Date pentru a calcula diferența dintre două date și pentru a actualiza un countdown timer.

    Schimbarea aleatoare a valorilor unei proprietăți: Este exemplificată prin schimbarea aleatoare a culorii conturului unui buton.

    Folosirea proprietăților classList, target sau currentTarget: Se folosește classList pentru a adăuga sau a elimina clase CSS dintr-un element.

    Folosirea metodelor getComputedStyle, stopPropagation: Nu este prezentă în codul furnizat.

    Validarea datelor dintr-un formular folosind expresii regulate: Nu este prezentă în codul furnizat.
